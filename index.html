<rockiot-gauge>

serial required [1] Gauge ID Has to be unique
variation required [radial] Gauge Type radial / linear
orientation [vertical] Gauge Orientation vertical,horizontal (applied only for linear gauges)
title [gauge] Gauge Title A title
units [null] Gauge Units Units (%, km/h, etc.
min [0] Min value can be set negative
max [100] Max value
value required [0] Current value
precision [0] Value decimal precision
size [md] Bar size sm , md , lg
radius [md] Radial radius size sm , md , lg
svgwidth [300] SVG width (px) You can set a custom width of the SVG. For linear vertical gauge width is set to 120px
svgheight [300] SVG height (px) You can set a custom height of the SVG. For linear vertical gauge width is set to 370px
scale [0] Display scale
smallscale [0] Display small scale
ticks [10] Number of ticks (divisions) on the scale Applied when scale=1
needle [0] Display gauge needle
animation [500] Animation interval (ms) 0 = no animation
bar-color [#444] Bar color
bar-border-color [#444] Bar border color
progress-color [#00ffff] Progress bar color (current value)
progress-border-color [#00ff00] Progress bar border color
needle-color [#444] Bar color
